# SuperNova-DravidianLangTech-ACL2026
Official implementation of Team SuperNova for the DravidianLangTech@ACL 2026 shared task on Abusive Tamil Text Targeting Women on Social Media. The repository includes three approaches: end-to-end fine-tuned MuRIL, MuRIL embeddings with XGBoost, and SentenceBERT with ensemble classifiers.
